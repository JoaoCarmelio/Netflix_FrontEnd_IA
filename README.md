# Netflix_FrontEnd_IA
Este projeto foi desenvolvido com o auxílio de Inteligência Artificial, sendo inspirado na interface da Netflix. A implementação contou com o suporte do GitHub Copilot.

## Objetivo
O objetivo desse projeto foi praticar o desenvolvimento front end com ajuda de ferramentas de IA.

## Tecnólogias usadas
- HTML
- CSS
- JAVASCRIPT
- GITHUB CAPILOT
- GEMINAI(Gereação de imagens)

## Como executar
1. Clone o repositório
2. Abra o arquivo index.html no navegador

## Como foi realizado
Este projeto foi inspirado na interface da Netflix e desenvolvido com o auxílio de Inteligência Artificial (GitHub Copilot).

A IA foi utilizada de forma estratégica para auxiliar na geração de código, sugestões de melhorias e correções. No entanto, também foram realizadas algumas implementações e ajustes manuais, garantindo melhor compreensão do código e maior controle sobre o resultado final.

Além disso, foram aplicadas boas práticas na criação de prompts, buscando obter respostas mais claras, eficientes e alinhadas com os objetivos do projeto.
